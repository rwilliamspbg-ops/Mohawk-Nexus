Architecture & Integration

Mohawk Nexus bridges two specialized worlds via a shared bridge contract. It integrates lightweight Go control planes for AF_XDP host integration and bridge request ingestion with highly optimized, memory-safe Rust datapaths.

Zero-Copy by Design

Operating entirely through AF_XDP kernel-bypass mechanisms, the Nexus minimizes memory allocations (averaging just 1-3 allocs/op in MRC spraying). By avoiding the traditional TCP/TLS kernel stack penalties, it delivers competitive speeds against standard DPDK setups, but with built-in formal safety verified in Lean 4.
