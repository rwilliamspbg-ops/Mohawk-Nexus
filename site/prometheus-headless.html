What This Headless View Proves

Direct API Querying Queries Prometheus over the HTTP API rather than relying on cached UI state.
Zero Interaction Loads automatically, refreshes on a timer, and exposes only the telemetry needed for operations.
Graceful Recovery Keeps the last successful values visible and marks fallback mode if the endpoint is unreachable.
Deployment Friendly Works as a kiosk page, dashboard embed, or hidden operator monitor in a larger app.
